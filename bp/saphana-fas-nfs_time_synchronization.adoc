= 時刻の同期
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
ストレージコントローラと SAP HANA データベースホストの間で時刻を同期する必要があります。そのためには、すべてのストレージコントローラとすべての SAP HANA ホストに同じタイムサーバを設定します。
